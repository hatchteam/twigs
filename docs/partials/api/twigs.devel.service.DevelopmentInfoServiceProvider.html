DevelopmentInfoServiceProvider
object in module twigs.devel

Description

Inject into your config function to set a url filter pattern.

App.config(function (DevelopmentInfoServiceProvider) {
  DevelopmentInfoServiceProvider.setUrlFilterPattern(/\/ws\/rest\//);
});

Methods

  • setUrlFilterPattern(pattern)

    Pass in a regex pattern (e.g. /\/ws\/rest\//) to filter XHTTP request urls. Most of the time, you only want to include REST request and exclude angular's request to templates etc.

    Parameters
    ParamTypeDetails
    patternstring

    A regex pattern to filter XHTTP Request urls
