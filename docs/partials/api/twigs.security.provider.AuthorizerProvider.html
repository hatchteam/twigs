  • registerUserLoader(loader)

    Registers the loader function to load the user Object. The given loader function must return a promise which resolves to the user Object. The user object is expected to be of the form:

     {
  username:'John',
  permissions:[]
 }

    It is valid to resolve to a user object which has additional properties.

    AuthorizerProvider.registerUserLoader(function ($q, $resource) {
      return function () {
          var deferred = $q.defer();
          $resource('/users/current').get({},
              function (data) {
                 return deferred.resolve(data);
             }, function () {
                 return deferred.reject();
              });

         return deferred.promise;
     };
  });
