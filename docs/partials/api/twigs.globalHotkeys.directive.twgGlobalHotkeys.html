twgGlobalHotkeys
directive in module twigs.globalHotkeys

Description

This directive registers for the 'keydown' event and forwards them to either custom hotkeys for the current page or global hotkey function, if defined.

You can use this directive on the html element, in order to catch all key events.

Ignores keystrokes in form elements , buttons and links.

See GlobalHotkeysService for more information.

Usage

as attribute
<ANY twg-global-hotkeys>
   ...
</ANY>
