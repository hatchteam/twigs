GlobalPopupsProvider
object in module twigs.globalPopups

Description

GlobalPopupsProvider can be used to define custom GlobalPopups or override twitch default GlobalPopups.

Usage

You need to specify a config block to create a new GlobalPopup and you need to create a html template for the Popup content.

angular.module('myApp').config(function (GlobalPopupsProvider) {
     GlobalPopupsProvider.createModal('myOwnPopup',{
         modalOptions: {
             templateUrl: 'views/globalMsg/myOwnPopup.html',
             windowClass: 'modal-myOwnPopup',
             backdrop: 'static',
             keyboard: false
         }
     });
});

Methods

  • createFileModal(messageName, options)

    Defines a File Modal. Can be used to display files directly in the browser, i.e. PDF. The Browsers file display is used.

    Parameters
    ParamTypeDetails
    messageNameString

    The name of this Modal, is later used to display this Modal with GlobalPopups.messageName('http://myurl.com', 'title', 'back');

    optionsObject

    conatining the options of this Modal. All options of UI Bootstrap Modals are additionally possible Required properties: * modalOptions: { templateUrl (required) specifying the location of the html template for this popup. }

    Example:

    GlobalPopupsProvider.createFileModal('fileDialog',{
     modalOptions: {
         templateUrl: 'templates/fileModal.html',
         windowClass:'modal-file',
         keyboard: true
     }
});

  • createToast(messageName, options)

    Defines a Toast.

    Parameters
    ParamTypeDetails
    messageNameString

    The name of this Toast, is later used to display this Modal with GlobalPopups.messageName('user saved successfully');

    optionsObject

    conatining the options of this Modal. Required property is 'templateUrl' specifying the location of the html template for this popup. All options of UI Bootstrap Modals are additionally possible Required properties: templateUrl (required) specifying the location of the html template for this popup. splayDuration (optional) specifying the timeout in miliseconds until the toast is hidden again.
